5. [Script](/Script) - Este diretório contém o script Python relacionado ao desafio.
6. [Agendamento](/Agendamento) - Contém informações e arquivos relacionados ao agendamento e scripts de inicialização.

Clique nos links acima para explorar as diferentes partes do desafio e acessar os arquivos e informações específicos de cada seção.
